# Factorie Clothing Store Website
A fully responsive, multi-page fashion retail website designed for the brand **Factorie Clothing Store**.  
The website features a modern aesthetic, clean navigation, a functional enquiry form, an image lightbox gallery, and embedded Google Maps for easy location viewing.

This project demonstrates practical front-end development skills using **HTML5, CSS3, and JavaScript**.
