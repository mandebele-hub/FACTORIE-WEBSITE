## Factorie Clothing Store Website
A responsive and stylish clothing store website built with **HTML**, **CSS**, and **JavaScript**.  
It showcases collections, allows customers to send enquiries, and provides contact information.

## SITEMAP
index.html # Home page
about us.html # About Us page
collections.html # Collections / Products page
enquiry.html # Enquiry form page
contact.html # Contact information + form
css/style.css # Main stylesheet
js/script.js # JavaScript for interactivity

## IMPROVEMENTS
A number of improvements were performed throughout the development process to raise the website's overall caliber.
To provide consistency across all pages, the layout was changed from a basic static structure to a responsive grid/flexbox design.
Images were calibrated to scale properly without disrupting the layout, and the navigation bar was enhanced to better fit smaller displays.
Hover effects were added to buttons and links to make the user experience more engaging, and typography and spacing were changed to make the text easier to read.
In order to improve usability and uniformity throughout the website, the footer was finally changed.
